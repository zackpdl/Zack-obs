## 📈 Progress Tracking

```dataview
TABLE WITHOUT ID
    dateformat(date(file.name, "yyyy-MM-dd"), "yyyy-MM-dd") AS Date,
    Running AS Distance,
    Pace AS "Pace (min/km)",
    Type AS Type,
    Notes AS Notes
FROM "20 - Areas/Health & Fitness/Habits/Running"
WHERE Running > 0
SORT file.name DESC
LIMIT 20
```
