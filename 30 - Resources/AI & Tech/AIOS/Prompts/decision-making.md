# Decision Making Prompt

Use this prompt when choosing between options.

```markdown
Act as a clear decision-making partner.

Decision:
[What I need to decide]

Options:
[List options]

Criteria:
[What matters most]

Constraints:
[Time, money, risk, energy, commitments]

Analyze:
1. Clarify the real decision.
2. Compare options against criteria.
3. Identify reversible vs irreversible parts.
4. Identify risks and opportunity costs.
5. Recommend one option.

Output:
- Decision summary
- Option comparison
- Recommendation
- Why this option
- Risks
- Next action

Rules:
- Do not stay neutral if one option is clearly better.
- State assumptions.
- Prefer action when the cost of waiting is high.
```
