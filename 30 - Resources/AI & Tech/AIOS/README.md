# AIOS

Portable AI Operating System layer for the ACE vault.

## Principle
Files over apps. Files over AI. Plain markdown over proprietary workflows.

AIOS is optional. The vault should remain useful without any AI tool.

## Structure
- `Maps`: identity and vault navigation.
- `Skills`: reusable workflows.
- `Prompts`: modular prompts for any AI.
- `Agents`: lightweight collaborator personas.
- `Templates`: structured session templates.

## Start Here
1. Read `AIOS/Maps/me.md`.
2. Read `AIOS/Maps/vault-map.md`.
3. Choose a skill, prompt, agent, or template based on the task.
4. Use the smallest useful context from the vault.
5. Save durable outputs into the correct ACE folder.

## Operating Rules
- Do not redesign ACE.
- Do not make AI the source of truth.
- Do not create fragile automations.
- Prefer links over tags.
- Prefer clear notes over complex metadata.
- Keep everything portable.
