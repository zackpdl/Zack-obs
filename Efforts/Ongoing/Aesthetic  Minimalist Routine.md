## 🏋️ Sample 3-Day “Aesthetic / Minimalist” Routine

This is a starting point. If it’s too easy or hard, adjust sets, reps, rest, or progression.

**Schedule suggestion**:

- Day A — Push + core
    
- Day B — Pull + (light legs)
    
- Day C — Legs + upper (balanced)
    
- Rest / recovery days in between
    

Each session ≈ 45–60 min, for ~3 total sessions per week → ~3 hours/week including warm-up, cool-down.

---

### Day A — Push / Upper Emphasis

1. **Barbell / Dumbbell Bench Press** (or push-up progression)
    
    - 3 sets × 6–10 reps
        
2. **Overhead Press** (standing dumbbell or barbell)
    
    - 3 × 6–10
        
3. **Incline Dumbbell Press / or Dumbbell Fly / Chest Isolation**
    
    - 2 × 8–12
        
4. **Lateral Raises (shoulder width aesthetic focus)**
    
    - 3 × 10–15
        
5. **Triceps Extension / Dips**
    
    - 2–3 × 8–12
        
6. **Core / Anti-extension work** (e.g. planks, hollow body, etc.)
    
    - 2–3 sets (maybe timed)
        

---

### Day B — Pull / Back & Light Leg Assistance

1. **Pull-ups / Chin-ups** (or progression / assistance)
    
    - 3 sets × as many good reps as possible
        
2. **Bent-over Row / Inverted Row / Dumbbell Row**
    
    - 3 × 6–10
        
3. **Face Pulls / Rear Delt / External Rotation work**
    
    - 2–3 × 12–15
        
4. **Hammer / Biceps Curl**
    
    - 2–3 × 8–12
        
5. **Hamstring / Posterior Chain work** (e.g. Romanian deadlift, glute-ham raises)
    
    - 2 × 8–12
        
6. **Optional core or grip work**
    

---

### Day C — Leg + Balanced Upper

1. **Squat variation** (back, front, goblet)
    
    - 3 × 6–10
        
2. **Lunges / Split Squats / Bulgarian Split Squats**
    
    - 2 × 8–12 per leg
        
3. **Calf Raise / Ankle Work**
    
    - 3 × 12–20
        
4. **Pull or Push accessory (alternate from Day A / B, whichever needs work)**
    
    - 2 sets
        
5. **Core / Midline stability**
    
    - 2–3 sets

## 🔄 Progression Strategy & Tracking

- Each workout, aim to **improve**: either more reps at the same load, or slightly heavier load if your form is solid.
    
- Log every set (weight, reps, rest) so you know when to push harder.
    
- Use “progression tiers” (e.g. once you can do 10 reps in all sets, increase load).
    
- If you stall, reduce volume or increase rest, then ramp back up.

## 🏠 Bodyweight / Minimal Equipment Version

If you don’t have gym or weights:

- Use **push-up progressions** (incline, weighted, archer, etc.)
    
- Use **pull-up / horizontal rows** with bars, rings, or TRX
    
- Single-leg work: **pistol squats, split squats, shrimp squats**
    
- Variations: **elevated feet, adding holds, tempo changes**
    
- Use progressions rather than just volume (i.e. harder variations over more reps)

## 🍎 Nutrition & Recovery (Brief Notes)

- To **lose fat and reveal shape**, you need a mild calorie deficit (eat slightly less than maintenance) while preserving protein intake.
    
- Don’t starve yourself — sustainable approach wins.
    
- Sleep, stress management, and consistency matter as much as training.
    
- Eat high-quality protein, veggies, and enough carbs/fats to support energy and recovery.

## 🛠 Tips & Customization

- If you're a **beginner**, start lighter, more reps, focus on form.
    
- If you feel drained, drop one isolation exercise rather than skip a main lift.
    
- After ~8–12 weeks, reassess — maybe swap variations, or push more intensity.
    
- Use deload weeks (lighter loads) if fatigue accumulates.
    
- If you have muscle imbalances or weak points, allocate optional “bonus” work carefully.
