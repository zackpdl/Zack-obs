Use [[Templates/Weekly Review]]. Clear Inbox, update Active Efforts, add missing contextual links, and choose next week priorities.

## Naming Convention Guide
- Evergreen notes: clean concept names, no dates. Example: [[Atlas/Notes/Knowing vs Doing]].
- MOCs: prefix with "MOC - ". Example: [[Atlas/Maps/MOC - Coding]].
- Calendar notes: use dates only. Example: [[Calendar/Daily/2026-05-16]].
- Efforts: use outcome names. Example: [[Efforts/Active/HalalHub]].
- Sources: use the book, article, video, or bookmark title.
- People: use human-readable names or group names.

## Lightweight Workflow
- Capture first in [[Inbox/Capture]].
- Process Inbox during daily or weekly review.
- Promote repeated clusters into MOCs only when they help retrieval.
- Use tags sparingly for states like #idea, #waiting, or #review.
