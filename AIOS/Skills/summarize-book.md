# Summarize Book

## Purpose
Convert a book, chapter, or reading note into durable knowledge and practical takeaways.

## Inputs
- Book title
- Author
- Notes, highlights, or chapter content
- Optional reason for reading

## Workflow
1. Identify the book's core argument.
2. Summarize major ideas by theme.
3. Extract useful models, quotes, or frameworks.
4. Connect ideas to Atlas notes or active Efforts.
5. Separate author claims from personal application.

## Output Format
```markdown
# Book Summary - Title

## Core Thesis

## Main Ideas
- 

## Useful Models
- 

## Applications
- 

## Related Notes
- 
```

## Rules
- Do not pretend to have read content not provided.
- Mark uncertain interpretations clearly.
- Prefer synthesis over chapter-by-chapter repetition.

## Example
Input: Highlights from a productivity book.
Output: Core thesis, 5 main ideas, practical applications, and links to relevant Efforts.
