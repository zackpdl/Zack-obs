# Daily Review

## Purpose
Close the day, reduce open loops, and connect daily context to Atlas and Efforts.

## Inputs
- Today's daily note
- Inbox items
- Active efforts
- Optional calendar/log context

## Workflow
1. Review today's focus and actual work.
2. Identify progress, blockers, and loose captures.
3. Move durable ideas to Atlas or link them.
4. Move outcome-oriented work to Efforts or link it.
5. Choose tomorrow's most important next step.

## Output Format
```markdown
## Daily Review

### Progress
- 

### Blockers
- 

### Captures To Process
- 

### Links Added
- 

### Tomorrow
- 
```

## Rules
- Keep it lightweight.
- Do not create new projects unless a real outcome exists.
- Prefer one clear next action over a long task list.

## Example
Input: Daily note plus Inbox capture.
Output: Summary, processed items, links to Efforts, and one next action.
