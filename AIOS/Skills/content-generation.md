# Content Generation

## Purpose
Create useful content from notes, ideas, or source material.

## Inputs
- Content goal
- Audience
- Platform
- Source notes
- Desired tone

## Workflow
1. Identify the audience and promise.
2. Extract the strongest idea.
3. Build a clear structure.
4. Draft in the requested format.
5. Improve clarity, retention, and usefulness.
6. Suggest titles or hooks if relevant.

## Output Format
```markdown
## Angle

## Structure

## Draft

## Optional Variants
- 
```

## Rules
- Stay true to the source material.
- Avoid fake urgency or fake expertise.
- Make the content clear, direct, and human.

## Example
Input: Notes on discipline and fitness.
Output: Short script, article outline, or post draft.
