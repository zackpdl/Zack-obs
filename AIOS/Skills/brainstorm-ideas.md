# Brainstorm Ideas

## Purpose
Generate useful ideas without losing strategic clarity.

## Inputs
- Topic or problem
- Goal
- Constraints
- Existing ideas or notes

## Workflow
1. Restate the goal.
2. Generate a broad set of options.
3. Cluster ideas by theme.
4. Evaluate by impact, effort, and fit.
5. Recommend the strongest next experiments.

## Output Format
```markdown
## Goal

## Ideas
- 

## Clusters
- 

## Best Bets
1. 
2. 
3. 

## Next Experiment
```

## Rules
- Favor actionable ideas over novelty.
- Avoid generic filler.
- Include constraints in evaluation.

## Example
Input: "Ideas for faceless AI YouTube videos."
Output: Clusters, high-retention angles, and a first experiment.
