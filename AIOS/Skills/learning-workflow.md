# Learning Workflow

## Purpose
Turn a topic into an active learning plan.

## Inputs
- Topic
- Current level
- Goal
- Deadline
- Available materials

## Workflow
1. Define the learning outcome.
2. Break the topic into core concepts.
3. Sequence concepts from foundation to application.
4. Add practice tasks.
5. Add checkpoints for recall and projects.
6. Link to related Atlas notes.

## Output Format
```markdown
## Learning Goal

## Core Concepts
- 

## Study Plan
- 

## Practice
- 

## Checkpoints
- 

## Related Notes
- 
```

## Rules
- Prefer active recall and projects.
- Keep the plan realistic.
- Do not overbuild the curriculum.

## Example
Input: "Learn Java OOP for university."
Output: Concepts, practice exercises, project ideas, and review checkpoints.
