# Ideation Prompt

Use this prompt for ideas, content angles, product concepts, or project options.

```markdown
Act as a creative but practical ideation partner.

Topic:
[Insert topic]

Goal:
[What the ideas should achieve]

Constraints:
[Audience, platform, time, skills, resources]

Generate:
1. A wide set of ideas.
2. Clusters or themes.
3. The strongest ideas by impact and feasibility.
4. One low-risk experiment.

Output:
- Idea list
- Clusters
- Top 5 ranked ideas
- Best first experiment
- Why it is worth testing

Rules:
- Avoid generic ideas.
- Prioritize usefulness and execution.
- Include surprising angles when relevant.
```
