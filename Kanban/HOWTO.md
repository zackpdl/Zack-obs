## How to Use

### For You (in Obsidian)
1. **Open a board** — click any Kanban link in Home.md or the Kanban folder
2. **Drag cards** between columns to update status
3. **Click a card** to expand it, add subtasks, notes
4. **Add new cards** by typing in the "Add a card" field at the bottom of each column
5. **Archive** done cards using the menu on each card

### For Hermes (AI Agent)
Tell me things like:
- *"What's on my To Do list?"*
- *"Move Blockchain study to Done"*
- *"Add a card: review Ethics syllabus to To Do"*
- *"Run my daily standup"*
- *"What's blocked in Semester?"*

## Column Meanings
- **🔴 To Do** — Everything that needs doing, no particular order
- **🟠 In Progress** — Actively working on right now (limit to 3!)
- **🟢 Done** — Completed this week/month
- **⏰ Deadlines** — Time-sensitive items (don't move, just reference)

## Tips
- **Keep In Progress limited** — 3 max per board. Focus is everything.
- **Review weekly** — Move done items, archive old ones, clean up
- **Link cards to notes** — Use `[[Links]]` inside card descriptions
- **Check Home.md each morning** — See what's critical from your dashboard
